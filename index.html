Projects

JobFair: The web app uses machine learning to help solve the problem of unconscious bias in the hiring process. It does this by anonymizing job applications via the analysis of text data to identify and recommend the removal of demographic signaling language. The app was developed using a Javascript frontend in ReactJS and a backend built in Express, which featured a Python script that utilized scikit learn for text analysis.
